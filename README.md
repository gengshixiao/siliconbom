# 硅宝 (SiliconBom)

一个专为电子工程师打造的AI助手平台，帮助您更高效地完成研发任务。

## 功能特性

- 🏠 **工作台** - 智能对话界面，支持多种研发模式
- 📚 **知识库** - 文档管理和智能问答
- 💬 **会话管理** - 历史会话记录和管理
- ⚙️ **个人设置** - 研发偏好和账户管理

## 技术栈

- 纯静态网站（HTML + CSS + JavaScript）
- 响应式设计
- 现代化UI/UX

## 项目结构

```
硅宝/
├── index.html              # 首页
├── knowledge-base.html     # 知识库页面
├── css/                    # 样式文件
│   ├── variables.css      # CSS变量
│   ├── common.css         # 公共样式
│   ├── sidebar.css        # 侧边栏样式
│   ├── index.css          # 首页样式
│   └── knowledge-base.css  # 知识库样式
└── js/                     # JavaScript文件
    ├── sidebar.js         # 侧边栏功能
    ├── index.js           # 首页逻辑
    ├── knowledge-base.js # 知识库逻辑
    └── ...
```

## 本地运行

直接在浏览器中打开 `index.html` 即可。

## 部署

### 使用 Nginx

1. 将项目文件上传到服务器
2. 配置 Nginx 指向项目目录
3. 访问网站

### 使用 GitHub Pages

1. 将代码推送到 GitHub
2. 在仓库设置中启用 GitHub Pages
3. 选择分支和目录
4. 访问 `https://YOUR_USERNAME.github.io/REPO_NAME`
