# 手动修复 "src refspec main does not match any" 错误

这个错误表示本地没有 `main` 分支或没有提交记录。

## 解决步骤：

在终端依次执行：

```bash
cd /Users/geng/Downloads/硅宝

# 1. 检查当前状态
git status
git branch

# 2. 添加所有文件
git add .

# 3. 创建提交（必须要有提交才能推送）
git commit -m "Initial commit: 硅宝项目"

# 4. 检查提交是否成功
git log --oneline

# 5. 创建并切换到main分支（如果还没有）
git branch -M main

# 6. 配置远程仓库
git remote remove origin 2>/dev/null
git remote add origin https://gengshixiao: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@github.com/gengshixiao/siliconbom.git

# 7. 推送
git push -u origin main
```

## 或者使用修复脚本：

```bash
chmod +x fix-and-push.sh
./fix-and-push.sh
```
