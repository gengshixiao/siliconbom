# 推送代码到GitHub - 完整步骤

## 第一步：初始化Git仓库并创建提交

在终端执行：

```bash
cd /Users/geng/Downloads/硅宝

# 初始化Git仓库
git init

# 添加所有文件
git add .

# 创建初始提交
git commit -m "Initial commit: 硅宝项目"
```

## 第二步：添加远程仓库

```bash
# 添加GitHub远程仓库
git remote add origin https://github.com/gengshixiao/siliconbom.git

# 检查远程仓库是否添加成功
git remote -v
```

## 第三步：推送到GitHub

```bash
# 设置主分支
git branch -M main

# 推送到GitHub（会提示输入用户名和密码/token）
git push -u origin main
```

## 如果提示输入用户名和密码：

1. **用户名**：输入 `gengshixiao`
2. **密码**：不能使用GitHub密码，需要使用 Personal Access Token

### 如何获取 Personal Access Token：

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. Token名称：输入 `siliconbom-deploy`
4. 勾选 `repo` 权限
5. 点击 "Generate token"
6. **复制生成的token**（只显示一次！）
7. 在终端密码提示处，粘贴这个token（而不是你的GitHub密码）

## 完成！

推送成功后，访问 https://github.com/gengshixiao/siliconbom 就能看到你的代码了！
